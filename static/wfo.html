Card Editor Guide

Drawing with hex bytes

  • Digit pairs drive drawing. Treat every two hexadecimal characters (for example 4F) as a single command on the 8×8 grid.
  • Bit layout matters. The byte splits into a xxx b yyy; the three-bit values xxx/yyy set horizontal/vertical offsets. Values 0–4 move forward, 5–7 wrap to −1, −2, −3.
  • Command bits pick the stroke. The outer bits a and b (written together as ab) choose the action: 00 draws a straight line, 01 and 10 draw quarter arcs when both axes move or half arcs when only one axis moves (vertical or horizontal bow), and 11 moves without drawing.
  • Zero moves are special. With no movement, 00 snaps to the row top, 01 plots a point, 10 draws a full circle, and 11 does nothing.

Path notation

Rules can describe strokes with arrow symbols instead of raw hex. Use the Keyboard palette to insert characters.

  • Counts. Optional move lengths use sans-serif bold digits 𝟬–𝟵 (default 1) so they never clash with hex bytes — e.g. ←𝟮, ↓𝟰.
  • Visible moves. ← left, → right, ↑ up, ↓ down.
  • Chaining & diagonals. Join segments with ⎹ (e.g. ←𝟮⎹↓𝟰). Combine axes in one token for diagonals (e.g. ←𝟮↑𝟯 or ←𝟵↑𝟵).
  • Invisible moves. ⮜ ⮞ ⮝ ⮟ — same counts as visible arrows (e.g. ⮞𝟯⎹⮝𝟭).
  • Arcs. ◖→𝟮↑𝟯 horizontal bow, ◗→𝟮↑𝟯 vertical bow.
  • Special marks. ⯭ snap, ∗ dot, ⍛ minicircle.
  • Shared vocabulary. Card 1.1 maps path tokens to hex; glyph cards usually end with {use_rule 1.1}.

Layout in rules & input

  • New row. ⌇⌇ (two ⌇) starts a new word or row.
  • Glyph delimiter. A single ⌇ separates successive glyphs in a rule or hex stream.
  • Comments. Prefix a line with //; the renderer ignores it. Literal hyphens in text stay as -.

Rules & includes

  • Replacements. Add pairs like 13,24 (source,target) in the Rule box. Pairs on the same line apply simultaneously; later lines cascade before glyphs are drawn. Use \, for a literal comma source.
  • Pull in another card. {use_rule 1.2} and {use_input 1.2} can appear anywhere in Rule or Input; nested tokens in the inserted text expand too. A cycle or missing card logs a warning and inserts nothing for that token.

SVG backgrounds

  • In the SVG field use {use_svg moon.svg} or just moon.svg (files live in public/svg, served at /svg/). You can also paste full SVG markup or use a remote URL.
